clean

约 96 字小于 1 分钟

2025-03-13

主要作用

移除工作区未被跟踪的文件

语法

git clean [-d] [-f] [-i] [-n] [-q] [-e <pattern>] [-x | -X] [--] <path>…​

常用的参数

参数解释
-d移除空目录
--f,--force强制删除

常用命令举例：

  • 移除新增的文件

    git clean --f .

  • 移除新增的文件和文件夹

    git clean -df .
