Sass 变量是存储值的容器，可以在整个样式表中重复使用，使代码更易于维护和修改。

## 变量声明

### 基本语法

```scss
$variable-name: value;
```

### 命名规则

- 以美元符号 `$` 开头
- 名称可以包含字母、数字、下划线和连字符
- 遵循 kebab-case（推荐）或 camelCase 命名约定

### 有效声明示例

```scss
// 颜色变量
$primary-color: #3498db;
$secondary-color: #2ecc71;

// 字体变量
$font-stack: "Helvetica Neue", Helvetica, Arial, sans-serif;
$base-font-size: 16px;

// 间距变量
$spacing-unit: 1rem;
$section-padding: $spacing-unit * 2;

// 布尔值
$dark-theme: true;
```

## 变量使用

### 基本使用

```scss
$brand-color: #4285f4;

.header {
  background-color: $brand-color;
}

.button {
  color: $brand-color;
  border: 1px solid $brand-color;
}
```

### 变量作用域

1. **全局变量**：在样式表顶层声明（不在任何规则内）

   ```scss
   $global-var: "I'm global";

   .class {
     content: $global-var; // 可访问
   }
   ```

2. **局部变量**：在规则内声明

   ```scss
   .container {
     $local-var: "I'm local";
     content: $local-var; // 可访问
   }

   .another-class {
     content: $local-var; // 错误！无法访问
   }
   ```

### 变量默认值 (!default)

当变量可能被其他文件覆盖时使用：

```scss
$theme-color: #f06 !default; // 仅当 $theme-color 未定义时生效

.header {
  color: $theme-color;
}
```

## 实际应用示例

### 主题系统

```scss
// 主题变量
$primary-color: #3498db;
$secondary-color: #2ecc71;
$text-color: #333;
$light-bg: #f9f9f9;

// 使用
body {
  color: $text-color;
  background: $light-bg;
}

.button {
  background: $primary-color;
  color: white;

  &.secondary {
    background: $secondary-color;
  }
}
```

### 响应式设计

```scss
// 断点变量
$breakpoint-sm: 576px;
$breakpoint-md: 768px;
$breakpoint-lg: 992px;

// 使用
@media (min-width: $breakpoint-md) {
  .container {
    width: 750px;
  }
}
```

### 数学运算

```scss
$base-spacing: 1rem;

.container {
  padding: $base-spacing;
}

.section {
  margin-bottom: $base-spacing * 2;
}
```

## 注意事项

1. **变量提升**：Sass 变量会被提升，可以在声明前使用
2. **覆盖规则**：后面的声明会覆盖前面的声明
3. **不可用于选择器名**：变量不能直接用于选择器名（需要使用插值 `#{$var}`）
4. **类型灵活**：变量可以存储任何 Sass 数据类型（颜色、字符串、数字、列表、映射等）

Sass 变量是创建可维护样式系统的强大工具，特别适合大型项目和设计系统的开发。
