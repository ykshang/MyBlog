在 CSS Grid 布局中实现 **上两栏、下三栏** 的布局，可以通过定义网格区域或调整网格单元格的跨度来实现。以下是几种实现方式及代码示例：

---

### **方法 1：使用 `grid-template-areas`（直观推荐）**

通过命名网格区域直接控制布局结构：

**效果**：

- 上方两栏（`top-left` 和 `top-right`）平分宽度。
- 下方三栏中，`bottom-1` 占据左侧全部高度，`bottom-2` 和 `bottom-3` 平分右侧。

### **方法 2：调整单元格跨度（灵活控制）**

通过 `grid-column` 和 `grid-row` 手动控制单元格跨度：

```html
<div class="container">
  <div class="item top-left">上左</div>
  <div class="item top-right">上右</div>
  <div class="item bottom-1">下1</div>
  <div class="item bottom-2">下2</div>
  <div class="item bottom-3">下3</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr; /* 两列等宽 */
  grid-auto-rows: 100px; /* 行高固定（可选） */
  gap: 10px;
}

/* 上方两栏 */
.top-left,
.top-right {
  grid-column: span 1; /* 各占一列 */
}

/* 下方三栏 */
.bottom-1 {
  grid-column: 1; /* 第一列 */
  grid-row: 2 / 4; /* 跨两行 */
}
.bottom-2 {
  grid-column: 2; /* 第二列 */
  grid-row: 2; /* 第二行 */
}
.bottom-3 {
  grid-column: 2; /* 第二列 */
  grid-row: 3; /* 第三行 */
}
```

**效果**：

- 上方两栏各占一列。
- 下方左侧一栏（`bottom-1`）跨两行高度，右侧两栏（`bottom-2` 和 `bottom-3`）垂直排列。

---

### **方法 3：嵌套 Grid 容器**

如果布局更复杂，可以嵌套 Grid：

```html
<div class="container">
  <div class="top">
    <div class="top-left">上左</div>
    <div class="top-right">上右</div>
  </div>
  <div class="bottom">
    <div class="bottom-item">下1</div>
    <div class="bottom-item">下2</div>
    <div class="bottom-item">下3</div>
  </div>
</div>
```

```css
.container {
  display: grid;
  grid-template-rows: auto auto;
  gap: 10px;
}

.top {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 10px;
}

.bottom {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 三列等宽 */
  gap: 10px;
}
```

---

### **关键点总结**

1. **`grid-template-areas`**：适合结构清晰的布局，直观易维护。
2. **单元格跨度**：通过 `grid-column` 和 `grid-row` 精细控制。
3. **嵌套 Grid**：适合复杂布局，但可能增加 HTML 层级。

根据实际需求选择最合适的方式！
