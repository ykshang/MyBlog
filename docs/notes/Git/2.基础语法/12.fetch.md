::: tip 主要作用
一般用来检出提交，更新各个分支的指针。  
大部分情况下，都是用 pull 来做更新操作，但是有个例外情况，有时候指针没有正常更新，需要 fetch 做更新操作
:::

## 语法

```bash
git fetch [<options>] [<repository> [<refspec>…​]]
git fetch [<options>] <group>
git fetch --multiple [<options>] [(<repository> | <group>)…​]
git fetch --all [<options>]
```

## 常用的参数

| 参数  | 解释                   |
| :---- | :--------------------- |
| --all | 更新各个分支的指针位置 |

## 常用命令举例：

- 更新全部分支，指针没有正常更新的时候，使用这个可以更新。

```bash
git fetch --all
## 或者
git fetch .
```

- 有些远端分支被删除了，拉取时，自动移除对应本地的过时分支

```bash
git fetch --prune
## 或者使用缩写
git fetch -p
## 或者配置全局，每次拉取时，自动移除过时分支
git config --global fetch.prune true
```
