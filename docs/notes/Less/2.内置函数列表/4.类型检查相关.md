## 1、`isnumber` 是否为数字

`isnumber` 函数用于检查一个值是否为数字。返回值为 `true` 或 `false`。

### 语法

```less
isnumber(value);
```

- `value`：要检查的数值或变量。

### 示例

```less
isnumber(#ff0);     // false
isnumber(blue);     // false
isnumber("string"); // false
isnumber(1234);     // true
isnumber(56px);     // true
isnumber(7.8%);     // true
isnumber(keyword);  // false
isnumber(url(...)); // false
```
