## 1、`color` 颜色转换

解析颜色值，将表示颜色的字符串转换为颜色对象。

### 语法

```less
color(string);
```

- `string`: 一个字符串，表示指定的颜色。

### 例子

```less
color("#aaa"); // '#aaa'
```

## 2、`image-size` 获取图片尺寸

根据文件获取图片尺寸。

### 语法

```less
image-size(string);
```

- `string`: 一个字符串，表示指定的文件。

### 例子

```less
image-size("file.png"); // '10px 10px'
```

::: tip
此函数需要在每个环境中实现。目前仅在 `node` 环境中可用。
:::
