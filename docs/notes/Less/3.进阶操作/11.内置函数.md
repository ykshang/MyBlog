以下是 LESS 内置函数的完整清单，按功能分组整理，并附上简要说明和示例：

## 一、颜色操作函数

| 函数                 | 说明                    | 示例                                  |
| -------------------- | ----------------------- | ------------------------------------- |
| `saturation(@color)` | 提取饱和度（0%-100%）   | `saturation(hsl(90,80%,50%))` → `80%` |
| `lightness(@color)`  | 提取亮度（0%-100%）     | `lightness(hsl(90,80%,50%))` → `50%`  |

## 六、其他实用函数

| 函数        | 说明                         | 示例                                |
| ----------- | ---------------------------- | ----------------------------------- |
| `default()` | 在 `when` 条件中表示默认情况 | `.mixin() when (default()) { ... }` |


LESS 的函数覆盖了 **颜色处理**、**数学运算**、**字符串操作**、**列表管理** 和 **类型检查** 等场景，能够显著提升样式表的动态性和可维护性。建议结合变量和混合宏（Mixin）灵活使用，以实现复杂的样式逻辑。
