以下是 LESS 内置函数的完整清单，按功能分组整理，并附上简要说明和示例：

## 一、颜色操作函数

### 1、颜色通道调整

| 函数                    | 说明                      | 示例                                                          |
| ----------------------- | ------------------------- | ------------------------------------------------------------- |
| `lighten(@color, %)`    | 增加亮度（0%-100%）       | `lighten(#336699, 20%)` → `#5c8db8`                           |
| `darken(@color, %)`     | 降低亮度（0%-100%）       | `darken(#336699, 20%)` → `#1a3d66`                            |
| `saturate(@color, %)`   | 增加饱和度                | `saturate(#369, 10%)` → `#2d6ba3`                             |
| `desaturate(@color, %)` | 降低饱和度                | `desaturate(#369, 10%)` → `#3f6a96`                           |
| `fadein(@color, %)`     | 减少透明度（更不透明）    | `fadein(rgba(51,102,153,0.5), 20%)` → `rgba(51,102,153,0.7)`  |
| `fadeout(@color, %)`    | 增加透明度（更透明）      | `fadeout(rgba(51,102,153,0.5), 20%)` → `rgba(51,102,153,0.3)` |
| `fade(@color, %)`       | 设置透明度（0%-100%）     | `fade(#336699, 50%)` → `rgba(51,102,153,0.5)`                 |
| `spin(@color, deg)`     | 调整色相（-360° 到 360°） | `spin(#f00, 90)` → `#80ff00`                                  |

### 2、颜色混合

| 函数                       | 说明                       | 示例                               |
| -------------------------- | -------------------------- | ---------------------------------- |
| `mix(@color1, @color2, %)` | 混合两种颜色（权重百分比） | `mix(#f00, #00f, 50%)` → `#800080` |
| `tint(@color, %)`          | 混合白色                   | `tint(#369, 50%)` → `#9bb4d1`      |
| `shade(@color, %)`         | 混合黑色                   | `shade(#369, 50%)` → `#1c344d`     |

### 3、颜色信息提取

| 函数                 | 说明                    | 示例                                  |
| -------------------- | ----------------------- | ------------------------------------- |
| `red(@color)`        | 提取红色通道值（0-255） | `red(#336699)` → `51`                 |
| `green(@color)`      | 提取绿色通道值（0-255） | `green(#336699)` → `102`              |
| `blue(@color)`       | 提取蓝色通道值（0-255） | `blue(#336699)` → `153`               |
| `alpha(@color)`      | 提取透明度（0-1）       | `alpha(rgba(51,102,153,0.5))` → `0.5` |
| `hue(@color)`        | 提取色相（0-360°）      | `hue(hsl(90,100%,50%))` → `90`        |
| `saturation(@color)` | 提取饱和度（0%-100%）   | `saturation(hsl(90,80%,50%))` → `80%` |
| `lightness(@color)`  | 提取亮度（0%-100%）     | `lightness(hsl(90,80%,50%))` → `50%`  |

## 二、数学函数

| 函数                  | 说明                 | 示例                      |
| --------------------- | -------------------- | ------------------------- |
| `ceil(@number)`       | 向上取整             | `ceil(2.1)` → `3`         |
| `floor(@number)`      | 向下取整             | `floor(2.9)` → `2`        |
| `round(@number)`      | 四舍五入             | `round(2.5)` → `3`        |
| `percentage(@number)` | 转换为百分比         | `percentage(0.5)` → `50%` |
| `min(@numbers...)`    | 返回最小值           | `min(3, 2, 5)` → `2`      |
| `max(@numbers...)`    | 返回最大值           | `max(3, 2, 5)` → `5`      |
| `abs(@number)`        | 绝对值               | `abs(-10)` → `10`         |
| `sqrt(@number)`       | 平方根               | `sqrt(25)` → `5`          |
| `pow(@base, @exp)`    | 幂运算（`base^exp`） | `pow(2, 3)` → `8`         |
| `mod(@a, @b)`         | 取模运算             | `mod(10, 3)` → `1`        |
| `sin(@angle)`         | 正弦（弧度制）       | `sin(1.047)` → `0.866`    |
| `cos(@angle)`         | 余弦（弧度制）       | `cos(0)` → `1`            |
| `tan(@angle)`         | 正切（弧度制）       | `tan(0.785)` → `1`        |

## 三、字符串函数

| 函数                                       | 说明                       | 示例                                     |
| ------------------------------------------ | -------------------------- | ---------------------------------------- |
| `escape(@string)`                          | URL 编码字符串             | `escape("a=1")` → `a%3D1`                |
| `e(@string)`                               | 去除字符串引号（CSS 转义） | `e("Arial")` → `Arial`                   |
| `replace(@string, @pattern, @replacement)` | 替换字符串                 | `replace("Hello", "e", "a")` → `"Hallo"` |

## 四、列表函数

| 函数                         | 说明                           | 示例                                   |
| ---------------------------- | ------------------------------ | -------------------------------------- |
| `length(@list)`              | 返回列表长度                   | `length(1px solid red)` → `3`          |
| `extract(@list, @index)`     | 提取列表中第 n 项（从 1 开始） | `extract(red green blue, 2)` → `green` |
| `range(@start, @end, @step)` | 生成数值序列                   | `range(1, 3, 1)` → `1 2 3`             |

LESS 本身不提供 map 或 each 函数，但可通过 递归 + `extract()` 来实现遍历。

## 五、类型检查函数

| 函数                   | 说明               | 示例                             |
| ---------------------- | ------------------ | -------------------------------- |
| `isnumber(@value)`     | 是否为数值         | `isnumber(10px)` → `true`        |
| `isstring(@value)`     | 是否为字符串       | `isstring("text")` → `true`      |
| `iscolor(@value)`      | 是否为颜色值       | `iscolor(#fff)` → `true`         |
| `iskeyword(@value)`    | 是否为 CSS 关键字  | `iskeyword(none)` → `true`       |
| `isurl(@value)`        | 是否为 URL         | `isurl(url("img.png"))` → `true` |
| `ispixel(@value)`      | 是否以 `px` 为单位 | `ispixel(10px)` → `true`         |
| `ispercentage(@value)` | 是否以 `%` 为单位  | `ispercentage(50%)` → `true`     |
| `isem(@value)`         | 是否以 `em` 为单位 | `isem(2em)` → `true`             |

## 六、其他实用函数

| 函数                      | 说明                         | 示例                                |
| ------------------------- | ---------------------------- | ----------------------------------- |
| `unit(@dimension, @unit)` | 修改或去除单位               | `unit(10px, em)` → `10em`           |
| `color(@string)`          | 解析字符串为颜色值           | `color("rgb(255,0,0)")` → `#ff0000` |
| `convert(@value, @unit)`  | 单位转换（如 `cm` 转 `px`）  | `convert(2cm, px)` → `75.59px`      |
| `default()`               | 在 `when` 条件中表示默认情况 | `.mixin() when (default()) { ... }` |

## 七、作用域与逻辑函数

```less
if(condition, value1, value2)
```

- `condition`：一个布尔表达式。
- `value1`：如果 `condition` 为==真==时返回的值。
- `value2`：如果 `condition` 为==假==时返回的值。

示例：

```less
@max: -10;
@w: 50px;
@h: 50px;
.box {
  width: if(@max > 0, @w, @w * 2);
  height: if(@max > 0, @w, @w * 2);
  border: 1px solid #000;
}
```

**编译结果**：

```css
.box {
  width: 100px;
  height: 100px;
  border: 1px solid #000;
}
```

## 八、示例

综合使用函数

```less
@primary: #336699;
@spacing: 10px 20px;

.box {
  background: lighten(@primary, 20%);
  margin: extract(@spacing, 1) round(3.6);
  border: if(iscolor(@primary), 1px solid @primary, none);
}
```

**编译结果**：

```css
.box {
  background: #5c8db8;
  margin: 10px 4px;
  border: 1px solid #336699;
}
```

## 八、总结

LESS 的函数覆盖了 **颜色处理**、**数学运算**、**字符串操作**、**列表管理** 和 **类型检查** 等场景，能够显著提升样式表的动态性和可维护性。建议结合变量和混合宏（Mixin）灵活使用，以实现复杂的样式逻辑。
