## 一、`ceil()`

向上取整，返回大于或等于给定数字的最小整数。

### 1、语法

```less
ceil(number)
```

- `number`: 要取整的数字。

### 2、例子

```less
ceil(2.4) // 3
```

## 二、`floor()`

向下取整，返回小于或等于给定数字的最大整数。

### 1、语法

```less
floor(number)
```

- `number`: 要取整的数字。

### 2、例子

```less
floor(2.6) // 2
```

## 三、`percentage()`

将一个值转换为百分比。

### 1、语法

```less
percentage(value)
```

- `value`: 要转换为百分比的值。

### 2、例子

```less
percentage(0.5) // 50%
```

## 四、`round()`

四舍五入，根据精度返回对应的数值。

### 1、语法

```less
round(number, decimalPlaces)
```

- `number`: 要四舍五入的数字。
- `decimalPlaces`: 可选的，要保留的小数位数。默认为 0。

### 2、例子

```less
round(2.4)  // 2
round(2.62)  // 3
round(2.5)  // 3
round(2.67, 1)  // 2.7
```

## 五、`sqrt()`

计算一个数的平方根。

### 1、语法

```less
sqrt(number)
```

- `number`: 要计算平方根的数字。

### 2、例子

```less
sqrt(25)  // 5
sqrt(18.6)  // 4.312771730569565
sqrt(25cm) // 5cm
sqrt(18.6%) // 4.312771730569565%
```

## 六、`abs()`

计算一个数的绝对值。保持单位不变。

### 1、语法

```less
abs(number)
```

- `number`: 要计算绝对值的数字。

### 2、例子

```less
abs(25cm) // 25cm
abs(-18.6%) // 18.6%
```

## 七、`sin()`

计算正弦函数。

### 1、语法

```less
sin(number)
```

- `number`: 要计算正弦值的数字。

### 2、例子

```less
// sine of 1 radian
sin(1); // 0.8414709848078965;
// sine of 1 degree
sin(1deg); // 0.01745240643728351;
// sine of 1 gradian
sin(1grad); // 0.015707317311820675;
```
