## 一、`ceil()`

向上取整，返回大于或等于给定数字的最小整数。

### 1、语法

```less
ceil(number)
```

### 2、例子

```less
ceil(2.4) // 3
```

## 二、`floor()`

向下取整，返回小于或等于给定数字的最大整数。

### 1、语法

```less
floor(number)
```

### 2、例子

```less
floor(2.6) // 2
```

## 三、`percentage()`

将一个值转换为百分比。

### 1、语法

```less
percentage(value)
```

### 2、例子

```less
percentage(0.5) // 50%
```

## 四、`round()`

四舍五入，根据精度返回对应的数值。

### 1、语法

```less
round(number, decimalPlaces)
```

- `number`: 要四舍五入的数字。
- `decimalPlaces`: 可选的，要保留的小数位数。默认为 0。

### 2、例子

```less
round(2.4)  // 2
round(2.62)  // 3
round(2.5)  // 3
round(2.67, 1)  // 2.7
```

## 五、`sqrt()`

计算一个数的平方根。

### 1、语法

```less
sqrt(number)
```

### 2、例子

```less
sqrt(25)  // 5
sqrt(18.6)  // 4.312771730569565
sqrt(25cm) // 5cm
sqrt(18.6%) // 4.312771730569565%
```

## 六、`abs()`

计算一个数的绝对值。保持单位不变。

### 1、语法

```less
abs(number)
```

### 2、例子

```less
abs(25cm) // 25cm
abs(-18.6%) // 18.6%
```
