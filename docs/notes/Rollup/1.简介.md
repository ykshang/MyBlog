## 什么是 Rollup

Rollup 是一个 JavaScript 模块打包器，专注于 ES 模块（ESM）的处理。它于 2015 年由 Rich Harris（没错，就是 Svelte 的创造者！）开发，主打简洁高效。

## 为什么选择 Rollup

Rollup 以其简洁的配置和高效的打包能力而闻名。它的主要优势包括：

- **高效的打包**：Rollup 利用了 ES 模块的静态分析特性，生成的包体积小，这是因为它有个杀手锏功能——==Tree Shaking==！它能自动移除未使用的代码。
- **配置简单**：相比 Webpack 复杂的配置，Rollup 配置文件简单易懂，新手友好！
- **输出格式灵活**：支持多种模块格式，包括 ESM、CommonJS、UMD 等。
- **生态系统繁荣**：Rollup 采用了基于插件的架构，并且拥有一个活跃的插件生态系统，方便开发者扩展功能。
- **适合库开发**：如果你在开发一个 JavaScript 库，Rollup 绝对是首选（Vue、React、Three.js 等都在用它）。
