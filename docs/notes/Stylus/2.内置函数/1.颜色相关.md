## 1、`red()` 红色分量

语法：`red(color[, value])`

返回给定 `color` 的红色分量，或将红色分量设置为可选的第二个 `value` 参数。

```styl
red(#c00) // => 204

red(#000, 255) // => #f00
```

## 2、`green()` 绿色分量

语法：`green(color[, value])`

返回给定 `color` 的绿色分量，或将绿色分量设置为可选的第二个 `value` 参数。

```styl
green(#0c0) // => 204

green(#000, 255) // => #0f0
```

## 3、`blue()` 蓝色分量

语法：`blue(color[, value])`

返回给定 `color` 的蓝色分量，或将蓝色分量设置为可选的第二个 `value` 参数。

```styl
blue(#00c) // => 204
blue(#000, 255) // => #00f
```

## 4、`alpha()` 透明度

语法：`alpha(color[, value])`

返回给定 `color` 的 alpha 分量，或将 alpha 分量设置为可选的第二个 `value` 参数。

```styl
alpha(#fff) // => 1
alpha(rgba(0,0,0,0.3)) // => 0.3
alpha(#fff, 0.5) // => rgba(255,255,255,0.5)
```

## 5、`dark()` 是否为深色

语法：`dark(color)`

检查 `color` 是否为深色。

```styl
dark(black) // => true
dark(#005716) // => true
dark(white) // => false
```

## 6、`light()` 是否为浅色

语法：`light(color)`

检查 `color` 是否为浅色。

```styl
light(black) // => false
light(#005716) // => false
light(white) // => true
```

## 7、`hue()` 色调

语法：`hue(color[, value])`
返回给定 `color` 的色调，或将色调分量设置为可选的第二个 `value` 参数。

```styl
hue(hsl(50deg, 100%, 80%)) // => 50deg
hue(#00c, 90deg) // => #6c0
```

## 8、`saturation()` 饱和度

语法：`saturation(color[, value])`

返回给定 `color` 的饱和度，或将饱和度分量设置为可选的第二个 `value` 参数。

```styl
saturation(hsl(50deg, 100%, 80%)) // => 100%
saturation(#00c, 50%) // => #339
```

## 9、`lightness()` 亮度

语法：`lightness(color[, value])`
返回给定 `color` 的亮度，或将亮度分量设置为可选的第二个 `value` 参数。

```styl
lightness(hsl(50deg, 100%, 80%)) // => 80%
lightness(#00c, 80%) // => #99f
```

## 10、`hsla()` HSLA 颜色

语法：`hsla(h, s, l[, a])`

用于创建或者转换 HSLA 颜色，其中 `h`、`s`、`l` 分别是色调、饱和度和亮度，`a` 是可选的 alpha 分量。

```styl
hsla(50deg, 100%, 80%, 0.5) // =>hsla(50deg, 100%, 80%, 0.5)
hsla(#ffcc00) // // =>hsla(50deg, 100%, 80%, 0.5)
```

## 11、`hsl()` HSL 颜色

语法：`hsl(h, s, l)`

用于创建或者转换 HSL 颜色，其中 `h`、`s`、`l` 分别是色调、饱和度和亮度。

```styl
hsl(10, 50, 30) // => HSLA
hsl(#ffcc00) // => HSLA
```

## rgba(color | r,g,b,a)[#](https://stylus.nodejs.cn/docs/bifs.html#rgbacolor--rgba)

从 r、g、b、a 通道返回 `RGBA`，或提供 `color` 来调整 alpha。

```
rgba(255,0,0,0.5)
// => rgba(255,0,0,0.5)

rgba(255,0,0,1)
// => #ff0000

rgba(#ffcc00, 0.5)
// rgba(255,204,0,0.5)
```

另外，Stylus 也支持 `#rgba` 和 `#rrggbbaa` 表示法：

```
#fc08
// => rgba(255,204,0,0.5)

#ffcc00ee
// => rgba(255,204,0,0.9)
```

## rgb(color | r,g,b)[#](https://stylus.nodejs.cn/docs/bifs.html#rgbcolor--rgb)

从 r、g、b 通道返回 `RGBA` 或投射到 `RGBA` 节点。

```
rgb(255,204,0)
// => #ffcc00

rgb(#fff)
// => #fff
```

## blend(top[, bottom])[#](https://stylus.nodejs.cn/docs/bifs.html#blendtop-bottom)

使用正常混合将给定的 `top` 颜色混合到 `bottom` 颜色上。`bottom` 参数是可选的，默认为 `#fff`。

```
blend(rgba(#FFF, 0.5), #000)
// => #808080

blend(rgba(#FFDE00,.42), #19C261)
// => #7ace38

blend(rgba(lime, 0.5), rgba(red, 0.25))
// => rgba(128,128,0,0.625)
```

## lighten(color, amount)[#](https://stylus.nodejs.cn/docs/bifs.html#lightencolor-amount)

将给定的 `color` 减轻 `amount`。

该函数对单位敏感；它支持百分比，例如，如下所示：

```
lighten(#2c2c2c, 30)
// => #787878

lighten(#2c2c2c, 30%)
// => #393939
```

## darken(color, amount)[#](https://stylus.nodejs.cn/docs/bifs.html#darkencolor-amount)

用 `amount` 使给定的 `color` 变暗。

该函数对单位敏感；它支持百分比，例如，如下所示：

```
darken(#D62828, 30)
// => #551010

darken(#D62828, 30%)
// => #961c1c
```

## desaturate(color, amount)[#](https://stylus.nodejs.cn/docs/bifs.html#desaturatecolor-amount)

通过 `amount` 降低给定 `color` 的饱和度。

```
desaturate(#f00, 40%)
// => #c33
```

## saturate(color, amount)[#](https://stylus.nodejs.cn/docs/bifs.html#saturatecolor-amount)

用 `amount` 使给定的 `color` 饱和。

```
saturate(#c33, 40%)
// => #f00
```

## complement(color)[#](https://stylus.nodejs.cn/docs/bifs.html#complementcolor)

给出互补色。相当于 `180deg` 旋转色调。

```
complement(#fd0cc7)
// => #0cfd42
```

## invert(color)[#](https://stylus.nodejs.cn/docs/bifs.html#invertcolor)

反转颜色。红色、绿色和蓝色值反转。不透明度被保留。

```
invert(#d62828)
// => #29d7d7
```

## spin(color, amount)[#](https://stylus.nodejs.cn/docs/bifs.html#spincolor-amount)

按 `amount` 旋转给定 `color` 的色调。

```
spin(#ff0000, 90deg)
// => #80ff00
```

## grayscale(color)[#](https://stylus.nodejs.cn/docs/bifs.html#grayscalecolor)

给出给定颜色的灰度等效值。相当于 `desaturate(color, 100%)`。

```
grayscale(#fd0cc7)
// => #0cfd42
```

## mix(color1, color2[, amount])[#](https://stylus.nodejs.cn/docs/bifs.html#mixcolor1-color2-amount)

按给定的数量混合两种颜色。`amount` 是可选的，默认为 `50%`。

```
mix(#000, #fff, 30%)
// => #b2b2b2
```

## tint(color, amount)[#](https://stylus.nodejs.cn/docs/bifs.html#tintcolor-amount)

将给定的颜色与白色混合。

```
tint(#fd0cc7,66%)
// => #feaceb
```

## shade(color, amount)[#](https://stylus.nodejs.cn/docs/bifs.html#shadecolor-amount)

将给定的颜色与黑色混合。

```
shade(#fd0cc7,66%)
// => #560443
```

## luminosity(color)[#](https://stylus.nodejs.cn/docs/bifs.html#luminositycolor)

返回给定 `color` 的 [相对亮度](https://www.w3.org/TR/WCAG20/#relativeluminancedef)。

```
luminosity(white)
// => 1

luminosity(#000)
// => 0

luminosity(red)
// => 0.2126
```

## contrast(top[, bottom])[#](https://stylus.nodejs.cn/docs/bifs.html#contrasttop-bottom)

返回 `top` 和 `bottom` 颜色之间的 [对比度](https://www.w3.org/TR/2008/REC-WCAG20-20081211/#contrast-ratiodef) 对象，基于 Lea Verou 的 [script](https://github.com/LeaVerou/contrast-ratio/blob/gh-pages/color.js#L108) 底层“[对比度](https://leaverou.github.io/contrast-ratio/)”工具。

第二个参数是可选的，默认为 `#fff`。

返回对象中的主键是 `ratio`，只有当 `bottom` 颜色为透明时，它还有 `min` 和 `max` 值，与 `ratio` 不同。在这种情况下，`error` 还包含误差范围。

```
contrast(#000, #fff).ratio
// => 21
contrast(#000, rgba(#FFF, 0.5))
// => { "ratio": "13.15;", "error": "-7.85", "min": "5.3", "max": "21" }
```

## transparentify(top[, bottom, alpha])[#](https://stylus.nodejs.cn/docs/bifs.html#transparentifytop-bottom-alpha)

返回给定 `top` 颜色的透明版本，就好像它是在给定 `bottom` 颜色上混合的（或者最接近它的颜色，如果可能的话）。

第二个参数是可选的，默认为 `#fff`。

第三个参数是可选的，并覆盖自动检测到的 alpha。

```
transparentify(#808080)
// => rgba(0,0,0,0.5)

transparentify(#414141, #000)
// => rgba(255,255,255,0.25)

transparentify(#91974C, #F34949, 0.5)
// => rgba(47,229,79,0.5)
```
