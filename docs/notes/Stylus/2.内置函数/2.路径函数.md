## basename(path[, ext])[#](https://stylus.nodejs.cn/docs/bifs.html#basenamepath-ext)

返回 `path` 的基本名称，（可选）删除 `ext` 扩展名。

```
basename('images/foo.png')
// => "foo.png"

basename('images/foo.png', '.png')
// => "foo"
```

## dirname(path)[#](https://stylus.nodejs.cn/docs/bifs.html#dirnamepath)

返回 `path` 的目录名。

```
dirname('images/foo.png')
// => "images"
```

## extname(path)[#](https://stylus.nodejs.cn/docs/bifs.html#extnamepath)

返回 `path` 的文件扩展名（包括点）。

```
extname('images/foo.png')
// => ".png"
```

## pathjoin(...)[#](https://stylus.nodejs.cn/docs/bifs.html#pathjoin)

执行路径连接。

```
pathjoin('images', 'foo.png')
// => "images/foo.png"

path = 'images/foo.png'
ext = extname(path)
pathjoin(dirname(path), basename(path, ext) + _thumb + ext)
// => 'images/foo_thumb.png'
```
