## 一、type

`type` 是 TypeScript 中的一个关键字，用于定义类型别名。它可以给任何类型起别名，包括基本类型、包装类型，甚至于交叉类型、联合类型、甚至字面量类型等。

类型别名有啥好处：

1. 类型别名可以简化复杂类型的定义，提高了代码可维护性
2. 类型别名可以使用语义性比较强的名称，大大提高了代码可读性
3. 一次定义，多处使用，减少代码冗余、提高代码复用率。
4. 类型别名可以通过交叉`&`扩展成一个新类型，可扩展性高。

### 1、基础类型

`type` 可以定义基础类型，如：

```ts
type User_Id = number; // number 类型
type User_Name = string; // string 类型
type User_isAdult = boolean; // boolean 类型

const user_id: User_Id = 1;
const user_name: User_Name = "John";
const user_isAdult: User_isAdult = true;
```

### 2、对象类型

`type` 可以定义对象类型，如：

```ts
type User = {
  id: number;
  name: string;
  isAdult: boolean;
};
const user: User = {
  id: 1,
  name: "John",
  isAdult: true,
};
```

```ts
type id = number; // number 类型
type name = string; // string 类型
type isAdult = boolean; // boolean 类型

type User = {
  id: number;
  name: name;
  isAdult: isAdult;
};
const user: User = {};
type address = string | null; // string 类型或 null 类型
type skill = string & any; // string 类型和 any 类型的交叉类型
type user_experience = {
  company: string;
  position: string;
};
interface user {
  id: user_id;
  name: user_name;
  isAdult: user_isAdult;
}
class user {
  id: user_id;
  name: user_name;
  isAdult: user_isAdult;
}
type man = user; // 给接口 interface 类型起别名
type woman = user; // 给类 class 类型起别名
type fun = (user: user) => user; // 给函数类型起别名
type User_Gender = "male" | "female"; // 字面量类型
```

3. 在定义变量、函数、参数、返回值时，可以使用类型别名去指定类型
4.
5. `type` 一个比较特殊的用法就是定义字面量类型，如 `type User_Gender = "male" | "female";`
6. 类型别名可以使用 `&` 扩展新的字段，形成新的类型，这也是交叉类型常见一个用法。
7. 类型别名无法重复定义，会报错。
8. 支持使用`?`来定义可选属性。

以下以一个比较综合的例子来说明 `type` 的用法：

```ts
// 定义用户姓名类型为 string
type User_Name: string;
// 定义用户年龄类型为 number
type User_Age: number;
// 定义用户的工作经验详情的类型为一个对象，别名为 Experience
type User_Experience: {
  company: string;
  position: string;
  startDate: Date;
  endDate: Date;
};

// 用户类型
type User = {
  name: User_Name;
  age: User_Age;
  isAdult: boolean;
  experience: Array<User_Experience>;
};
const user: User = {
  name: "Alice",
  age: 25,
};
```
