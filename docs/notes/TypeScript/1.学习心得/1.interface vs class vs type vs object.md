首先我们先来介绍一下这几个类型的主要功能和特点：

### 1、接口（Interfaces）

1. 接口可以继承接口
2. 接口可以继承类
3. 接口可以继承对象

---

1. 接口用来描述对象的结构和方法。
2. 接口可以用来描述变量、函数入参的具体约束。
3. 接口可以被类实现，实现接口的类必须实现接口中定义的所有属性和方法。
4. 接口可以继承其他接口，实现多继承的效果。
5. 接口可以继承类的成员定义，但是不能继承类的具体实现
6. 接口可以继承接揭露

### 2、类（Classes）

1. 类可以继承类：一个类只能继承一个类，
2. 类不可以继承接口
3. 类不可以继承对象
4. 类可以实现接口
5. 一个类可以实现多个接口

---

1. 类可以实现接口，但是不能继承接口
2. 对象可以被接口继承，但是不能被类继承
