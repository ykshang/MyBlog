TypeScript 是 JavaScript 的超集，也就是说，TypeScript 包含了 JavaScript 的所有功能。
二者最大的不同就是，TypeScript 在 JavaScript 的基础上添加了一套类型系统。
这个类型系统的类型检查是静态的，也就是说，在编译时就会进行类型检查。
基于这个类型系统，提供了一系列的关于类型的特性，例如：类型运算、类型推断、类型兼容性、类型别名等。
基于类型系统，提出了接口、类、抽象类、泛型的概念。
