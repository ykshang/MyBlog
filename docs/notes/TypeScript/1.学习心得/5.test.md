# TypeScript 中的 Class 全面介绍

TypeScript 对 ES6 的类进行了扩展，添加了类型注解和其他面向对象编程特性。下面是 TypeScript 中 class 的详细介绍：

## 1. 基本类结构

```typescript
class Person {
  // 属性声明（可以带类型注解）
  name: string;
  age: number;
  
  // 构造函数
  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }
  
  // 方法
  greet() {
    console.log(`Hello, my name is ${this.name}`);
  }
}

const person = new Person("Alice", 30);
person.greet();
```

## 2. 访问修饰符

TypeScript 提供了三种访问修饰符：

```typescript
class Example {
  public publicProp: string;    // 默认，任何地方可访问
  private privateProp: string;  // 仅类内部可访问
  protected protectedProp: string; // 类及其子类可访问
  
  constructor(public readonly id: number) {
    // 参数属性 - 直接声明并初始化成员
  }
}
```

## 3. 继承

```typescript
class Animal {
  constructor(public name: string) {}
  
  move(distance: number = 0) {
    console.log(`${this.name} moved ${distance}m`);
  }
}

class Dog extends Animal {
  constructor(name: string, public breed: string) {
    super(name); // 必须调用父类构造函数
  }
  
  bark() {
    console.log("Woof! Woof!");
  }
  
  // 方法重写
  move(distance = 5) {
    console.log("Running...");
    super.move(distance);
  }
}
```

## 4. 抽象类

```typescript
abstract class Shape {
  abstract area(): number; // 抽象方法，必须由子类实现
  
  // 可以有具体实现
  displayArea() {
    console.log(`Area: ${this.area()}`);
  }
}

class Circle extends Shape {
  constructor(public radius: number) {
    super();
  }
  
  area(): number {
    return Math.PI * this.radius ** 2;
  }
}
```

## 5. 静态成员

```typescript
class Utilities {
  static PI = 3.14159;
  
  static calculateArea(radius: number): number {
    return this.PI * radius * radius;
  }
}

console.log(Utilities.calculateArea(5));
```

## 6. 接口实现

```typescript
interface Loggable {
  log(message: string): void;
}

class Logger implements Loggable {
  log(message: string) {
    console.log(`[LOG] ${new Date().toISOString()}: ${message}`);
  }
}
```

## 7. 存取器 (Getters/Setters)

```typescript
class Temperature {
  private _celsius: number = 0;
  
  get celsius(): number {
    return this._celsius;
  }
  
  set celsius(value: number) {
    if (value < -273.15) {
      throw new Error("Temperature below absolute zero!");
    }
    this._celsius = value;
  }
  
  get fahrenheit(): number {
    return this._celsius * 9/5 + 32;
  }
  
  set fahrenheit(value: number) {
    this._celsius = (value - 32) * 5/9;
  }
}
```

## 8. 类表达式

```typescript
const Person = class {
  constructor(public name: string) {}
};

const person = new Person("Alice");
```

## 9. 高级类型：类作为类型

```typescript
class Point {
  constructor(public x: number, public y: number) {}
}

// 类可以作为类型使用
function printPoint(p: Point) {
  console.log(p.x, p.y);
}

// 也可以用于 instanceof 检查
if (something instanceof Point) {
  // ...
}
```

## 10. 装饰器 (实验性特性)

```typescript
// 需要启用 experimentalDecorators
function sealed(constructor: Function) {
  Object.seal(constructor);
  Object.seal(constructor.prototype);
}

@sealed
class Greeter {
  greeting: string;
  
  constructor(message: string) {
    this.greeting = message;
  }
  
  greet() {
    return "Hello, " + this.greeting;
  }
}
```

## 11. 泛型类

```typescript
class Box<T> {
  private contents: T;
  
  constructor(value: T) {
    this.contents = value;
  }
  
  getValue(): T {
    return this.contents;
  }
}

const numberBox = new Box<number>(42);
const stringBox = new Box<string>("Hello");
```

## 12. this 类型

```typescript
class Calculator {
  value: number;
  
  constructor(value = 0) {
    this.value = value;
  }
  
  add(n: number): this {
    this.value += n;
    return this; // 支持链式调用
  }
  
  multiply(n: number): this {
    this.value *= n;
    return this;
  }
}

new Calculator(1).add(2).multiply(3).value; // 9
```

## 类与接口的区别

| 特性        | 类 (Class)               | 接口 (Interface)         |
|------------|--------------------------|--------------------------|
| 运行时存在   | 是                       | 否 (编译时概念)          |
| 可实例化    | 是                       | 否                       |
| 可包含实现   | 是                       | 仅声明                   |
| 可扩展      | 通过继承                 | 通过继承或合并           |
| 类型检查用   | 可以                     | 主要用途                 |

TypeScript 的类提供了强大的面向对象编程能力，同时保持了类型安全。它们既可以在运行时使用，也可以作为类型系统的一部分。
