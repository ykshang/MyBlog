## 一、简介

### 1、声明方式

#### 1. 推荐方式

使用`class`关键词来声明一个类。

```ts
class Point {
  x: number;
  y: number;
}
```

#### 2. 匿名类

使用变量来声明一个类。

```ts
let Point = class {
  x: number;
  y: number;
};
```

#### 3. 二者区别

二者的区别：

- 类表达式可以立即执行，类声明不行。
- 类表达式可以继承自一个类或实现一个接口，类声明不行。
- 类表达式可以有静态成员，类声明不行。
