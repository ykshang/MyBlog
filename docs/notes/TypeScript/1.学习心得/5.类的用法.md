## 一、简介

### 1、声明方式

我们一般使用`class`关键词来声明一个类。

```ts
// 显式类
class Point {
  x: number;
  y: number;
  constructor(x: number, y: number) {
    this.x = x;
    this.y = y;
  }
}
```

### 2、类的成员

类的成员主要以下三种：属性、方法、构造器。此外，还有一些特性，其实都是基于这三种成员提供的扩展功能。

我们在定义类的成员时，需要给出具体实现。

```ts
class Point {
  x: number;
  y: number;
  constructor(x: number, y: number) {
    this.x = x;
    this.y = y;
  }
  sepak() {
    console.log("Hello, World!");
  }
}
```

其实方法也是一种特殊的属性，只不过它的类型是函数类型。

如果使用以下写法，需要在实例化的时候，给出具体的实现。

```ts
class Demo {
  sepak: () => void;
  constructor(sepak: () => void) {
    this.sepak = sepak;
  }
}
let fun_param = () => {
  console.log("Hello, World!");
};
let point = new Demo(fun_param);
```

## 二、高级特性

### 1、存取器（`getter`/`setter`）

存取器（`accessor`）是特殊的类方法，包括取值器（`getter`）和存值器（`setter`）两种方法。

取值器用来读取属性，存值器用来写入属性：

```ts
class C {
  _prop = "";
  get prop() {
    return this._prop;
  }
  set prop(value) {
    this._prop = value;
  }
}
let c = new C();
c.prop = "123"; // 调用 setter
console.log(c.prop); // 调用 getter
```

还有一种写法：

```ts
class DynamicProperties {
  private _data: { [key: string]: any } = {};

  // 索引 getter
  get [key: string]: any {
    return this._data[key];
  }

  // 索引 setter
  set [key: string](value: any) {
    this._data[key] = value;
  }
}

const obj = new DynamicProperties();
obj['name'] = 'Alice';  // 调用索引 setter
console.log(obj['name']); // 调用索引 getter，输出 'Alice'
```

### 8、匿名类（类表达式）

匿名类是指没有名称的类。一般使用类表达式来定义匿名类。

```ts
// 匿名类
let Point = class {
  x: number;
  y: number;
};
```

**二者区别：**

| **特性**   | **显式类**                                 | **匿名类**                 |
| ---------- | ------------------------------------------ | -------------------------- |
| 类名       | 有正式类名（如 Person）                    | 无正式类名，只有变量名     |
| 类型系统   | 类名即类型（Person 类型）                  | 结构类型（通过变量名引用） |
| 可访问性   | 可在定义前引用（hoisting）                 | 必须先定义后使用           |
| 调试信息   | 显示实际类名                               | 显示为 class 或变量名      |
| 继承       | 可直接继承（class Student extends Person） | 只能通过变量名继承         |
| 导出/导入  | 可直接导出类                               | 只能导出变量               |
| 构造函数名 | 与类名相同                                 | 显示为 "class"             |
