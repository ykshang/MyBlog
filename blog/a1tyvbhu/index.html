2025-9-17 联通面试复盘

  1. Vue2 和 Vue3 响应式的区别？分别怎么实现的？
  2. vue2的响应式有什么缺陷？怎么解决？（vue.$set、重新封装）
  3. vue3的响应式存在哪些问题？又该解决？（问的兼容性、babel、polyfill）
  4. 万行数据的虚拟列表怎么实现？
    1. 真分页、假分页
    2. 只显示可视区域、Observation API 监控，调整滚动条位置
    3. 现成的包
  5. lock文件干啥
  6. 如何封装一个用于万行数据的虚拟列表的组件
    1. 适配深浅色主题、要不要支持响应式
    2. 导出形式：CJS、CMS、UEM
    3. 引用形式：公共组件、npm包、pnpm子包
  7. 了解哪些响应式的解决方案，详细介绍一下
    1. 媒体查询
    2. vw、vh
    3. rem
    4. 断点怎么选择，了解过一些主流框架怎么实现响应式方案吗？bootstarp、unocss
  8. package.json配置里export、type字段都是干啥的。对应 6.2 往下问的
  9. 浏览器从输入网址到页面渲染完成的全过程
  10. 知道webkit和gecko的区别吗？
  11. js文件的加载时机了解过吗？
    1. defer、async
    2. webpack打包出来的是defer
  12. local storage、sessionstorage、indexdb的区别。
  13. http缓存、网络缓存
    1. 强缓存、协商缓存
    2. etag、那几个相关请求头都干啥的
    3. 文件指纹
  14. webpack构建慢咋优化
    1. 代码分割
    2. 关闭sourcemap
    3. 多线程
  15. 文件过大怎么解决
    1. 手动拆包，调整limit
    2. 标记副作用、尽量写纯函数组件
    3. webpac 那几个魔法注释
