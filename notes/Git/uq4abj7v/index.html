各种骚操作

约 198 字小于 1 分钟

2025-02-16

快速切换到上一个分支

  git checkout -
  git swotch -

快速清空本地修改。

  • 恢复到和某个远端的分支一致，使用reset

    git reset -hard 远端/分支名

  • 清空工作区未提交的修改，使用clean

    git clean -df

拉出分支

## 基于某个提交拉出DTS分支
git branch DTS b54d17b
## 基于某个tag拉出DTS分支
git branch DTS V1.1.1

配置别名实现快速操作

  ## 以图形结构显示日志
  git config --global alias.lg "log -graph"
  ## 查看本地和远端全部的分支
  git config  --global alias.bra "branch -a"
  ## 查看远端全部的分支
  git config  --global alias.brr "branch -r"
  ## 修改分支描述
  git config  --global alias.bre "branch --edit-description"

使用：

  git lg
  git bra
  git brr
  git bre branch_name
